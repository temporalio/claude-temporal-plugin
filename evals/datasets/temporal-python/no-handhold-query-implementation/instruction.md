Create a Temporal workflow in Python that tracks a counter using signals and queries.

Implement this in counter.py with a workflow named "CounterWorkflow" that supports an "increment" signal and a "get_count" query. The main block should demonstrate incrementing and querying.

Running `uv run python counter.py` should show the counter going from 0 to 3.

The temporal dev server is available on this system, but you may need to add dependencies to this project.

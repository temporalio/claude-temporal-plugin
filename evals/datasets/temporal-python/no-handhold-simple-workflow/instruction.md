Create a greeting service using Temporal in Python that says hello to a given name.

Implement this in greeting.py with a workflow named "GreetingWorkflow" and a runnable main block that accepts a name as a CLI argument and prints the greeting.

Running `uv run python greeting.py John` should print `Hello, John!`.

The temporal dev server is available on this system, but you may need to add dependencies to this project.

Create a Temporal workflow in Python that handles cancellation gracefully.

Implement this in cancellable.py with a workflow named "CancellableWorkflow" that iterates with delays, catches cancellation, and reports how many iterations completed.

Running `uv run python cancellable.py` should print something like `Cancelled after N iterations`.

The temporal dev server is available on this system, but you may need to add dependencies to this project.
